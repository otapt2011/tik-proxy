TikTok Proxy API

This API provides access to public TikTok profile data, statistics, and user videos without any official TikTok SDK. It scrapes the public HTML of TikTok profiles and extracts embedded JSON data.

All endpoints require an API key sent via the X-API-Key or Authorization header.

Base URL: https://tik-proxy.vercel.app

Authentication Details

Every API request must include a valid API key. Provide it via:

  • X-API-Key: your_secret_key
  • Authorization: your_secret_key

Set API_SECRET_KEY in Vercel environment variables.

# Example header
X-API-Key: sk_live_abc123def456
Disclaimer: This service is for educational purposes only. Please respect TikTok’s terms of service and use responsibly.

Available Endpoints

GET
/api/full?username=...
Complete profile + stats (avatar, bio, verified, follower/following counts, heart count, video count).
Required: username query param (e.g., ?username=tiktok)
{"success":true,"profile":{"username":"tiktok","displayName":"TikTok"},"stats":{"followerCount":12345678}}
GET
/api/tiktok?username=...
Basic profile: username, display name, avatar URL, followers, following, total likes, total videos.
Required: username query param (e.g., ?username=tiktok)
{"username":"tiktok","display_name":"TikTok","followers":12345678}
GET
/api/tiktokFull?username=...
Raw full user detail object from TikTok’s embedded data.
Required: username query param (e.g., ?username=tiktok)
{"userInfo":{"user":{"uniqueId":"tiktok"},"stats":{...}}}
GET
/api/stats?username=...
Only stats: followerCount, followingCount, heartCount, videoCount, diggCount.
Required: username query param (e.g., ?username=tiktok)
{"followerCount":12345678,"followingCount":123,"heartCount":987654321}
GET
/api/user/[username]
Nested route returns user detail JSON.
Required: username in path (e.g., /api/user/therock)
{"userInfo":{"user":{"uniqueId":"therock"},"stats":{...}}}
GET
/api/user/[username]/stats
Stats from nested route (alternative to /api/stats).
Required: username in path (e.g., /api/user/therock/stats)
{"followerCount":12345678,"followingCount":123}
GET
/api/user/[username]/videos?limit=12
Fetch user videos (up to 50). Returns video URLs, cover, stats, hashtags.
Required: username in path; optional: limit query (e.g., /api/user/tiktok/videos?limit=5)
{"success":true,"videos":[{"id":"123","playCount":1000000}]}
POST
/api/batch
Batch fetch multiple profiles. Body: { "usernames": ["user1","user2"] }.
Required: JSON body with array "usernames" (e.g., { "usernames": ["tiktok", "therock"] })
{"success":true,"results":[{"username":"tiktok","success":true}]}
